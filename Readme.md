#### Proyecto backend

**express**
```
npm i express
```
**nodemon como dependencia de desarrollo**
```
npm i --save-dev nodemon
```
**mongoose**
```
npm i mongoose
```

**dotenv sirve para realizar varibles de entorno**
```
npm i dotenv
```
**bcrypt permite cachear los password**
```
npm i bcrypt
```
**jsonwebtoken**
```
npm i jsonwebtoken
```
**cors**
```
npm i cors
```
**nodemailer**
```
npm i nodemailer
```
**Se levanta el banckend**
```
npm i start
```
